## Game of Drones
Backend based on the game rock, paper or scissors.

## Tools and versions used

- NPM version 6.4.1
- NodeJS version 10.13.0
- MongoDB version 4.2.0
- Express version 4.17.1
- Express Async Errors version 3.1.1
- Joi version 14.3.1
- Joi ObjectId version 2.0.0
- Lodash version 4.17.15
- Mongoose version 5.7.1
- Winston version 3.2.1
- Jest version 24.9.0

## Available Scripts

In the project directory, you can run:

### `node index.js or nodemon index.js`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

**Remember to have NPM packages downloaded before running this project locally**

### `npm test`

Runs unit tests for the entire project with help of Jest library.
